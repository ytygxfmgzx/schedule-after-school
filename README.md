# 兴趣班排课小助手

> 🌐 **在线体验**：<https://schedule-after-school.pages.dev/>

一个基于纯 HTML/CSS/JavaScript 的兴趣班课程排课工具，帮助家长或老师管理和查看兴趣班课程安排。

## 功能特性

- **课程管理** — 添加、编辑、删除兴趣班课程，支持自定义颜色和教练信息
- **日历视图** — 以月历形式展示课程安排，支持切换"全部课程"与"仅已排课"视图
- **冲突检测** — 自动检测同一时间段的课程冲突，并在日历上醒目标记
- **单节课控制** — 右键点击日历上的课程，可单独取消或恢复某节课
- **课程开关** — 在侧边栏点击课程卡片可快速启用/禁用整门课程的显示
- **数据导入/导出** — 支持将课程数据导出为 JSON 文件，也可从文件导入
- **图片导出** — 一键将当前日历导出为 PNG 图片
- **恢复排课** — 一键恢复所有未排（取消）的课
- **清空数据** — 一键清空全部课程数据
- **响应式布局** — 支持桌面端和移动端自适应显示

## 使用方式

最简单的方式是直接访问上方「在线体验」地址，开箱即用。

若需本地运行，直接在浏览器中打开 `index.html` 即可，无需任何构建步骤或服务器：

```
# 或使用任意静态文件服务器
python -m http.server 8000
```

## 技术栈

- HTML5 / CSS3 / JavaScript（ES6+）
- Material Design 3 风格
- localStorage 本地持久化存储
- [html2canvas](https://html2canvas.hertzen.com/) 用于日历图片导出
- [Google Fonts](https://fonts.google.com/)（Noto Sans SC、Material Symbols）
